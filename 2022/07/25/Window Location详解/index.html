Window Location 详解

2022-07-25
前端性能

window.location 对象用于获得当前页面的地址 (URL)，并把浏览器重定向到新的页面，在编写时可不使用 window 这个前缀。

// 整个URl字符串(完整的URL)
// https://blog.zhuhaiting.com/about/?q=front-end#one
location.href;

// 协议部分
// https:
location.protocol;

// 主机部分
// blog.zhuhaiting.com
location.hostname;

// 端口号
// 80，没有为空
location.port;

// 站点主地址
// https://blog.zhuhaiting.com
location.origin;

// 主机名和当前URL的端口号
// blog.zhuhaiting.com
location.host;

// 路径部分(文件地址)
// /about/
location.pathname;

// 查询部分——从问号？开始的URL
//  ?q=front-end
location.search;

//锚部分 #号后面
// #one
location.hash;

// assign——加载新的文档
// location.assign("https://www.baidu.com")
location.assign();

// reload——重新加载当前文档
// location.reload(true) 默认 false——客户端缓存里取当前页，true——服务端取最新的页面, 相当于客户端点击 F5("刷新")
location.reload();

// replace——用新的文档替换当前文档,无浏览历史纪录,不能使用“前进”和“后退”
// location.replace("https://www.baidu.com")
location.replace();
